# 古风言情提词器

本文档是 AI 写作古风言情的核心提词器。按章节结构分段，每段有独立的提词模板和约束条件。

---

## 使用方法

将以下提词模板复制到 AI 对话中，按步骤生成初稿。每一步生成后，先检查再进入下一步。

---

## 第一章提词模板

### Step 1：开头钩子（3-5 句）

```
请用以下约束写一个古风言情小说的开头（3-5 句）：

【约束】
1. 第一句话必须是感官刺激或动作，不要环境描写
2. 不要用"她睁开眼睛"、"这是一个…"开头
3. 不要用任何 AI 高频词（不禁、竟然、五味杂陈、心中一紧）
4. 必须在前 3 句内建立悬念或冲突
5. 感官描写至少涉及 2 种（视/听/嗅/味/触）
6. 比喻必须来自角色的生活经验，不要通用意象

【题材】{genre}
【主角】{character_name}，{character_type}，当前处境：{situation}

【开篇思考方法——不要套用模板，按以下逻辑自己推导】

第一步：确定"醒来"的方式
- 主角处于什么状态？（睡觉/昏迷/发呆/干活/赶路/被关押/受伤/生病/喝酒/等死……）
- 用什么感官刺激把他/她"唤醒"或"激活"？（疼痛/寒冷/声音/气味/触碰/光线/晃动/窒息/口渴/饥饿……）
- 这个感官刺激要跟主角当前处境相关（被追杀→疼痛；穷困→饥饿；深宅→压抑的声音；战场→爆炸声）

第二步：确定"第一眼看到什么"
- 主角睁开眼/回过神后，视线先落在什么东西上？
- 这个东西要能暗示处境（破庙→穷困；闺房→深宅；刑场→危险；喜堂→婚礼）
- 不要写"看到了房间"——要写具体的一样东西（一只碗/一道裂纹/一扇门/一个人的背影）

第三步：确定"第一个动作"
- 主角做的第一个有意识的动作是什么？
- 这个动作要能展示性格（谨慎的人会先观察；冲动的人会立刻起身；绝望的人会继续躺着）
- 动作要具体，不要"她坐起来"——要"她撑着胳膊坐起来，骨头嘎吱响"

第四步：确定"悬念钩子"
- 前3句内要让读者产生疑问
- 疑问可以来自：（为什么醒来？/这是哪里？/为什么疼？/谁在说话？/这是什么时候？/为什么他/她会在这里？）
- 不要在第一段就回答这个疑问——留着

【禁止套用的模式】
- 不要用"X是被X醒的"开头——太常见了
- 不要用"不是X，是X"的否定句式开头——太模板化
- 不要用老槐树/破庙/青砖地等固定意象——除非题材必须
- 第一段不要超过5句——短而有力

请根据以上思考方法，为你的主角推导出一个独特的开篇，然后生成：
```

### Step 2：环境与身份确认（10-15 句）

```
继续上文，写主角确认当前处境的段落（10-15 句）：

【约束】
1. 不要用"她意识到自己重生了"——要通过感官细节让读者自己判断
2. 环境描写要服务于情绪，不要为写而写
3. 至少加入 1 个闲笔（跟情节无关但增加真实感的细节）
4. 身体感受要具体——不要"浑身酸痛"，要具体到哪个部位、什么感觉
5. 时间/空间要有具体锚点（几时了、初几、哪个院子/哪个房间/哪条街）

【时间】{time_period}
【地点】{location}
【需要确认的信息】{info_to_confirm}

【环境描写思考方法——不要套用模板，按以下逻辑自己推导】

第一步：确定"这个角色会注意什么"
- 不同身份的人，同一场景会注意不同的东西
- 穷人进房间先看有没有吃的；富家小姐先看有没有镜子；将军先看有没有出口
- 主角是什么出身？什么职业？什么性格？他/她会先注意什么？

第二步：确定"这个空间的独特细节"
- 不要写"房间里有桌椅"——太泛
- 要写这个房间独有的东西（一道裂缝/一个缺口/一块污渍/一个摆件/一种气味）
- 这个细节最好能暗示时间/历史/主人性格（碗沿缺口→用久了/穷；铜镜背面磨亮→习惯性动作；墙角蛛网→没人打扫）

第三步：确定"身份确认的方式"
- 主角怎么确认自己回到了什么时候？
- 不要直接写"她看到了自己的手"——太常见
- 想想其他方式：（听到熟悉的声音/闻到熟悉的气味/看到某个还没发生的事/摸到某个还没坏的东西/遇到某个还没死的人）

第四步：确定"情绪的物理表达"
- 主角震惊/困惑/害怕/愤怒时，身体会有什么反应？
- 不要写"她感到震惊"——要写具体反应
- 想想：（手指发抖/呼吸变浅/后背发凉/瞳孔放大/心跳加快/喉咙发紧/膝盖发软）

【禁止套用的模式】
- 不要用铜镜照脸——太常见
- 不要用"X岁的手"——太模板化
- 不要用碗沿缺口——太像示例
- 不要用老槐树/猫叫/货郎叫卖——除非你的场景真的有这些东西

请根据以上思考方法，为你的主角和场景推导出独特的环境描写，然后生成：
```

### Step 6：章末悬念（3-5 句）

```
继续上文，写本章结尾（3-5 句）：

【约束】
1. 不要总结本章——要留未完成的事
2. 结尾要有新的问题或新的威胁
3. 最后一句话要有画面感——不要抽象
4. 不要用"新的一天开始了"、"她决定好好规划未来"结尾
5. 可以用物件、声音、动作结尾

【结尾示例——不要套用，理解原理】

好的结尾有几种类型，选一种适合你的：

类型A：物件收尾
- 原理：用一个具体的物件结束，让读者去想这个物件意味着什么
- 例："她把那封信塞进袖子里。"（信里写了什么？她要拿它做什么？）
- 例："他盯着那枚铜钱看了很久。"（铜钱有什么特别？）

类型B：声音收尾
- 原理：用一个突然出现的声音结束，制造紧张感
- 例："门外传来脚步声。"（谁来了？来做什么？）
- 例："远处传来敲锣的声音。"（发生了什么？）

类型C：动作收尾
- 原理：用一个意味深长的动作结束，暗示接下来会发生什么
- 例："她把茶碗放下，站起来。"（她要去做什么？）
- 例："他转身往门外走。"（他要去哪？）

类型D：时间收尾
- 原理：用时间标记结束，暗示事情即将到来
- 例："帖子三天后才来。"（帖子来了会怎样？）
- 例："离端阳宴还有七天。"（端阳宴会发生什么？）

【思考步骤】
1. 本章解决了什么问题？→ 留一个没解决的
2. 本章建立了什么悬念？→ 让它更紧迫
3. 主角接下来要做什么？→ 用一个动作/物件暗示
4. 有什么威胁在逼近？→ 用声音/时间暗示

【需要埋下的伏笔】{foreshadow}
【新的威胁/问题】{new_threat}

请根据以上思考方法，为你的故事推导出独特的结尾，然后生成：
```

## 通用提词约束

每次使用提词器时，都必须在提词中附加以下约束：

```
【全局约束 — 必须遵守】
1. 禁用词：不禁、竟然、五味杂陈、百感交集、心中一紧、不由得、不由自主、
   目光深邃、嘴角微微上扬、微微一笑、温润如玉、静水深流、春日暖阳、
   倾国倾城、沉鱼落雁、闭月羞花、国色天香、风华绝代、仿佛（每段最多1个）
2. 比喻必须来自角色的生活经验，不要用通用意象
3. 每段至少 1 个感官细节（视/听/嗅/味/触）
4. 段落长度要有变化——不要每段都差不多长
5. 对白要有潜台词——角色说的话≠角色想的话
6. 情感不要直说——用身体反应展示
7. 每章至少 2 个闲笔（跟情节无关的细节）
8. 每章至少 1 处直接讲述（"她恨他"、"这事儿荒唐"）
9. 结尾不要总结——留悬念
10. 时间标记不要太精确——"过了几天"比"三天后"好
```

## 场景感官清单

| 场景 | 必须涉及的感官 |
|------|---------------|

## 五大写作技法

### 技法一：五感注入法

**原则**：每个场景至少涉及三种感官（视、听、嗅、味、触）。

**改写示例**：

```
❌ AI 原文：
他走进房间，感到一阵紧张。房间里很安静，他不禁有些害怕。

✅ 改写后：
他推开门，门轴发出一声闷响。空气里有股潮湿的霉味，混着旧报纸的油墨气息。
他的手指不自觉地攥紧了门把手，指节发白。天花板上的灯管嗡嗡响着，忽明忽暗。
```

**操作规则**：
- "感到 X" → 写出身体的具体反应
- "很安静" → 写出安静中能听到什么（自己的心跳、远处的车声、地板的吱呀）
- "很冷" → 写出冷的具体感受（鼻尖发麻、呼出白气、手指僵硬）

### 技法二：对白去模板化

**原则**：每个角色必须有独特的说话方式。

**对白规则**：
- 禁止角色说他们不会说的话（教授不会说"那可不"，出租车司机不会说"此言差矣"）
- 对白中加入打断、犹豫、未说完的句子
- 用动作替代"XX说"：他把烟摁灭了，"不去了。"
- 潜台词比明说更有力：角色说"没事"的时候，往往有事

**改写示例**：

```
❌ AI 原文（所有人说话都一样）：
张三说："我认为这件事情的处理方式需要重新考虑。"
李四说："我同意你的看法，这确实是一个需要认真对待的问题。"

✅ 改写后（角色语言差异化）：
张三说："这事儿不对劲，得重新合计合计。"
李四点了根烟，半天才开口："你说得对，可问题是——谁去说？"
```

### 技法三：节奏变奏法

**原则**：打破 AI 的均匀节奏，制造快慢交替。

| 场景类型 | 句子 | 段落 | 详略 |
|----------|------|------|------|
| 紧张/打斗 | 短句、碎句 | 短段落（1-3句） | 详写，放慢时间 |
| 日常/过渡 | 中等长度 | 中等段落 | 略写，一笔带过 |
| 回忆/沉思 | 长句、绵延 | 长段落 | 详写，沉浸感 |
| 惊讶/转折 | 突然短下来 | 单句成段 | 详写，定格瞬间 |

**改写示例**：

```
❌ AI 原文（节奏均匀）：
他走到了门口，犹豫了一下，还是推开了门。房间里很暗，他摸索着找到了灯的开关。
灯亮了，他看到了房间里的景象，不禁大吃一惊。房间里一片狼藉，到处都是散落的文件。

✅ 改写后（节奏变奏）：
他走到门口。
手搭上门把手，又放下了。
再搭上。
推开门。暗的。他摸到开关，啪嗒——
灯亮了。
他愣在原地。
文件散了一地。桌子翻了。窗帘被扯掉一半，露出后面被凿穿的墙壁。
```

### 技法四：流水账→场景化

**原则**：把"然后……然后……然后……"变成有画面、有冲突、有感官的场景。

**场景化检查清单**：
- 有没有具体的地点描写？
- 有没有人物的具体动作（不是"他做了XX"，而是怎么做的）？
- 有没有感官细节（看到/听到/闻到/触到什么）？
- 有没有冲突或张力（哪怕很小）？
- 有没有删除"然后"、"接着"、"随后"？

### 技法五：逻辑链补全

**原则**：AI 常见的逻辑断层必须补上因果链。

| 断层类型 | 表现 | 补法 |
|----------|------|------|
| 动机断层 | 角色突然做某事，没有铺垫 | 补上前文伏笔或内心活动 |
| 情感断层 | 情绪突变，没有过渡 | 补上情绪渐变过程 |
| 时间断层 | "三天后"突然出现，没有交代 | 补上中间发生了什么 |
| 空间断层 | 角色突然换了地方，没有移动描写 | 补上转移过程 |
| 信息断层 | 角色突然知道某事，没有信息来源 | 补上谁告诉他的/怎么发现的 |

## 深层结构去 AI 化

### 七大高级 AI 模式

#### 模式一：结构可预测性

**问题**：AI 的叙事结构像清单打勾。
**改法**：在预期的"A→B"之间插入不相关的场景（闲笔）；让某个场景同时承担两个矛盾的功能；打乱因果链。

#### 模式二：比喻安全区

**问题**：AI 的比喻永远"正确"——意象经典、逻辑自洽、读者秒懂。
**改法**：用角色职业/生活经验中的事物做比喻；允许比喻"不完美"；减少明喻，改用暗喻。

#### 模式三：信息密度均匀

**问题**：AI 的每一段都传递大致相同的信息量。
**改法**：重要场景拆成多个段落；过渡场景一句话带过；回忆段要有碎片感。

#### 模式四：无闲笔

**问题**：AI 的每个细节都有功能。
**改法**：每章加入 2-4 个与情节无关的细节；允许环境描写"跑题"；让角色有不重要的小习惯。

#### 模式五：Show don't tell 过度执行

**问题**：AI 被训练成"永远展示，不要讲述"，执行得太彻底。
**改法**：在关键情感爆发点直接讲述；允许叙述者偶尔评论；紧张场景中直接讲述可以加速节奏。

#### 模式六：情感节拍精准

**问题**：AI 的情感转折像节拍器一样精准。
**改法**：让角色在不恰当的时候有不恰当的情绪；情感爆发不要铺垫太足；让情感有"毛边"。

#### 模式七：缺乏作者个人癖好

**问题**：AI 没有写作习惯。
**改法**：确定 2-3 个个人癖好贯穿全文（偏好破折号、短句收尾、方言口语等）。

### 闲笔类型

| 类型 | 示例 | 作用 |
|------|------|------|
| 环境闲笔 | "树上有个鸟窝，鸟窝里有三颗蛋" | 让空间更真实 |
| 角色闲笔 | "她喝茶前总要吹三下" | 让角色更立体 |
| 叙述者闲笔 | "这事儿说起来挺荒唐" | 让叙述者有存在感 |
| 感官闲笔 | "空气里有股潮湿的霉味" | 让读者有沉浸感 |
| 时间闲笔 | "那只猫在墙头蹲了大概一炷香的工夫" | 让时间有质感 |

### 留白类型

| 类型 | 示例 |
|------|------|
| 情感留白 | "他走了。她站在门口，看着他的背影越来越小。"（不写她难不难过） |
| 对话留白 | ""我其实——"她的话卡在嗓子里，最终什么也没说。" |
| 结局留白 | "她把信折好，塞进袖子里。门外的脚步声越来越近。" |
| 细节留白 | "碗沿有一道细细的缺口。"（不解释为什么） |

## 语言指纹模板

为每部作品确定一个"语言指纹"——一组贯穿全文的语言习惯：

```markdown
## 语言指纹

### 标点习惯
- 破折号使用频率：高（每千字 ≥ 3 个）
- 省略号使用：只用于对白，叙述中不用
- 感叹号：极少使用（每章 ≤ 2 个）

### 句式习惯
- 收尾句偏好：短句收尾（3-7 字）
- 段首词偏好：避免重复，但允许"她"连续出现 2 次
- 长句上限：35 字

### 用词偏好
- 偏好用的连接词：可、倒是、偏偏
- 避免的连接词：然而、因此、同时
- 偏好用的动词：拽、摁、盯、攥
- 避免的动词：做、干、弄、搞

### 语气波动
- 回忆段：冷淡、克制
- 当下段：略带嘲讽
- 对白段：口语化、有打断
```

## 快速改写公式

当不确定怎么改时，套用以下公式：

### "感到 X" → 身体反应

```
感到紧张 → 手指不自觉地攥紧了衣角
感到害怕 → 后背一阵发凉，汗毛竖了起来
感到高兴 → 嘴角怎么也压不下去
感到悲伤 → 鼻子一酸，眼眶发烫
感到愤怒 → 牙关咬得咯咯响，太阳穴突突地跳
```

### "他做了X" → 场景展开

```
他吃了饭 → 他扒了两口饭，米粒硬邦邦的，菜凉了，油花凝成白色的膜
他走了过去 → 他踩着碎石子走过去，鞋底嘎吱嘎吱响，路灯把影子拉得老长
她哭了 → 她没出声，只是肩膀一抖一抖的，眼泪掉在手背上，温热的
```

### "然后" → 场景切换

```
删除"然后"，用以下方式替代：
1. 直接切场景（空一行）
2. 用时间标记："三天后"、"入秋的时候"
3. 用空间标记："回到出租屋"、"推开酒吧的门"
4. 用感官过渡："电话响了"、"门铃响了"、"雨停了"
```
